# 一个仿照掘金社区风格的web端博客项目

## 本地运行

   ```
   git clone https://github.com/zhaokgithub/venus-web.git

   cd venus-web

   npm install

   npm run serve

   npm run build
   ```
## 功能架构
   * 首页文章展示页面
   * 文章分类页面
   * 关于我页面
   * 写文章页面
   * 个人中心页面
